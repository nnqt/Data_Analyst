Investigation Overview

The primary goal of this investigation is to explore and analyze the FordGoBike dataset to uncover key patterns and insights about user behavior, trip characteristics, and potential relationships between different features. This exploration is aimed at understanding the dynamics of bike-sharing usage, such as the preferences of different user groups, trip durations, and variations based on demographics and user types.

Dataset Overview

The FordGoBike dataset provides detailed information about individual bike-sharing trips made in the San Francisco Bay Area. Key features of the dataset include:

  1. Trip Information:
  • duration_sec: The duration of each trip in seconds.
  • start_time and end_time: Timestamps for when trips started and ended.
  • start_station_id and end_station_id: IDs of the start and end stations for each trip.
  1. User Demographics:
  • member_birth_year: The year of birth of the user.
  • member_gender: The gender of the user (Male, Female, or Other).
  • user_type: The type of user (Subscriber or Customer).
  1. Station Information:
  • start_station_latitude and start_station_longitude: Geographic coordinates of the start station.
  • end_station_latitude and end_station_longitude: Geographic coordinates of the end station.

Dataset Overview

The FordGoBike dataset contains detailed information about individual bike trips taken in the San Francisco Bay Area. Each trip includes key attributes such as trip duration, start and end station locations, user type, gender, and birth year. These details provide an opportunity to analyze user behavior and identify usage trends in bike-sharing.

The dataset is well-suited for this investigation because it captures:

  1. Temporal data (start and end times) for analyzing patterns over time.
  2. Demographic information (gender and birth year) for studying user diversity.
  3. Trip-related metrics (duration and station coordinates) for uncovering mobility trends.

Key Findings and Insights

Subscribers tend to have shorter and more consistent trip durations compared to Customers, who exhibit greater variability.

Generated visualization shows shorter trip duration for Subscribers compared to Customers with clear log-scale adjustments for better visibility.

Age and Trip Duration Relationship

Older users generally take shorter trips, while younger users tend to have longer trips. This trend highlights how age influences bike usage.

Conclusion

The FordGoBike analysis revealed key insights into user behavior and trip patterns. Subscribers typically take shorter, more consistent trips, while Customers exhibit greater variability. Age and gender significantly influence trip duration, with younger users and male users being the most active. Additionally, trips are concentrated in urban, high-traffic areas, highlighting the system's role in city commuting. These findings provide valuable insights into user demographics and preferences, offering opportunities to improve bike-sharing services.
